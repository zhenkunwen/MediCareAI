## 项目概述

MediCareAI-Agent 是一个多 Agent 自主医疗协作系统。核心定位：**不是聊天机器人，而是由多个医疗智能体组成的自主协作系统**。

### 生产环境 (Production)

- 域名 (Domain): `https://openmedicareagent.online`
- VPS: `root@47.108.235.30`
- 容器 (Containers): 8 (backend, frontend, postgres, redis, celery-worker, celery-beat, searxng, valkey)

### 快速开始 (Quick Start)

```bash
git clone https://github.com/HougeLangley/MediCareAI-Agent.git
cd MediCareAI-Agent
cp .env.example .env
docker compose up -d
# API: http://localhost:8000
# 接口文档: http://localhost:8000/docs
```

### 开发工作流

```
本地开发 → git push → VPS git pull → docker compose build → docker compose up -d
```

> ⚠️ 标准部署流程：Backend 先构建，Frontend 后构建，禁止并行。详见 [deployment](./deployment.mdx)。
